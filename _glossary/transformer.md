**Transformer** 是一种基于自注意力机制的深度神经网络架构，2017 年由 Google 在论文 "Attention Is All You Need" 中提出。它摒弃了传统的循环/卷积结构，完全依赖注意力机制建模序列数据中的长距离依赖关系，奠定了现代大语言模型的基础。

## 架构概览

```
输出序列
    ▲
    │
┌───┴──────────┐
│  输出投影      │
└───┬──────────┘
    │
┌───┴──────────┐
│  Add & Norm   │◀── 残差连接 + 层归一化
│  前馈网络      │
└───┬──────────┘
    │
┌───┴──────────┐
│  Add & Norm   │
│  Multi-Head   │
│  Attention    │
└───┬──────────┘
    │
    │  + Positional Encoding
    │
输入序列
```

## 核心组件

- **自注意力（Self-Attention）**：每个位置关注所有位置，计算加权表示
- **多头注意力**：在多个子空间并行计算注意力
- **位置编码**：为序列注入位置信息
- **前馈网络**：逐位置的非线性变换
- **残差连接**：稳定深层网络训练
