# 一、整体项目背景
项目：基于多模态数据融合的**口腔种植术后边缘骨吸收风险预测**
- 数据：口腔影像 + 临床结构化数据（多模态）
- 任务：二分类风险预测（高风险/低风险），属于**样本不均衡医疗任务**
- 核心方案：ResNet50 提取影像特征 + 临床变量引导乘法融合策略
- 实验范式：**患者级分层划分 + 5折交叉验证**，规避数据泄露

---

# 二、网络基础相关
## 1. FC 全连接层
1. 定义：Fully Connected Layer，CNN 网络末端的线性层。
2. 原生作用：将卷积/池化输出的高维特征，映射为分类标签输出。
3. 项目用法：**去掉FC层**，截断分类头，把 ResNet 当作特征提取器，输出 2048 维图像特征，用于后续多模态融合。

## 2. 多模态融合策略
- 采用**临床结构化变量引导的乘法融合**。
- 对比：拼接融合、加法融合易产生冗余特征；乘法融合可利用临床信息动态增强图像特征，适配本任务场景。

# 三、模型有效性验证（核心：判断模型是否学到真实特征）
## 1. 标签洗牌实验
1. 做法：仅随机打乱**训练集**风险标签，图像与临床数据保持不变，重新训练。
2. 判定标准（只看**验证集/测试集**指标，不看训练集）：
   - 洗牌后 AUC≈0.5：模型学到数据中**真实风险特征**，无捷径学习；
   - 洗牌后指标依旧偏高：模型学习到数据偏差、噪声，属于**捷径学习**。
3. 补充：训练集天然可拟合乱标签，无参考价值。

## 2. 综合验证手段（整套评估体系）
1. 数据层面：患者级数据集划分 + 5折交叉验证 + 外部独立测试集，防泄露、过拟合；
2. 消融实验：单模态/多模态对比、影像ROI遮挡、临床变量剔除，验证各模块真实贡献；
3. 可解释性分析：Grad-CAM热力图（观察注意力是否聚焦病灶）、SHAP特征归因（量化变量贡献）；
4. 鲁棒性测试：数据增广、扰动测试、分布外样本测试。

## 3. 模型概率输出解读
1. 计算流程：网络FC层输出 Logit → Sigmoid 归一化为 0~1 概率值。
2. 含义：概率代表**模型置信度**，不等同于临床真实风险；人为设定阈值后，才划分成二分类标签。
3. 现存问题：深度网络普遍存在**过度自信、置信度失准**；分为偶然不确定性（数据噪声）、认知不确定性（模型能力不足）。
4. 优化方案：温度缩放等概率校准方法。

# 四、生存分析补充模型：Cox 比例风险模型
## 1. 基础介绍
经典**半参数生存分析模型**，适配医疗随访数据，可处理**截尾数据**（随访未发病、失访样本）。
公式：$h(t|X) = h_0(t) \cdot \exp(\sum\beta_i X_i)$
- $h_0(t)$：基准风险（无需预设分布）；$\beta$：协变量系数；$HR=\exp(\beta)$ 风险比。

## 2. 核心前提
**比例风险假设**：不同样本风险比值不随时间变化，用 Schoenfeld 残差检验验证。

## 3. 评价指标
- C指数：衡量事件发生顺序预测能力（生存任务核心）；
- 校准曲线、AIC、似然比检验。

## 4. 与 CNN 二分类模型对比
| 维度 | Cox模型 | CNN多模态分类模型 |
| ---- | ---- | ---- |
| 任务 | 生存分析、预测事件发生时间+瞬时风险 | 静态二分类、固定时间点判断风险 |
| 数据 | 支持截尾随访数据 | 不原生支持截尾数据 |
| 可解释性 | 强（HR、系数可临床解读） | 弱，依赖事后可解释方法 |
| 特征能力 | 擅长低维临床变量 | 擅长高维影像、挖掘非线性特征 |

## 5. 结合项目用法
ResNet 提取影像特征，和临床变量共同作为协变量输入 Cox 模型，兼顾特征提取与临床预后分析。

# 五、分类模型评价指标（混淆矩阵为根基）
## 基础定义
$TP$真正例、$FP$假正例、$TN$真负例、$FN$假负例；
阳性=高风险患者，阴性=低风险患者。

## 核心指标（公式+含义+场景）
1. **TPR 真正例率 / Recall 召回率**
   $TPR=\dfrac{TP}{TP+FN}$
   真实阳性样本被检出比例，衡量**漏诊**；ROC曲线纵轴。

2. **FPR 假正例率**
   $FPR=\dfrac{FP}{FP+TN}$
   真实阴性样本被误判比例，衡量**误诊**；ROC曲线横轴。

3. **PPV 阳性预测值 / Precision 精确率**
   $PPV=\dfrac{TP}{TP+FP}$
   模型预测阳性结果的可信度；PR曲线纵轴。

4. **NPV 阴性预测值**
   $NPV=\dfrac{TN}{TN+FN}$
   模型预测阴性结果的可信度。

> 补充：无标准缩写 PPR、NPR，属于笔误，统一记 PPV、NPV。

5. **Accuracy 准确率**
   全局预测正确率，**样本不均衡场景（本项目）基本不使用**。

6. **F1分数**
   $F1=\dfrac{2\cdot PPV \cdot Recall}{PPV+Recall}$
   PPV与Recall的调和平均，均衡两项指标，不均衡数据首选。

## 曲线指标
1. **ROC曲线 & AUC**
   横轴FPR、纵轴TPR；AUC为曲线下面积，表征模型整体区分能力。
2. **PR曲线 & AUPRC**
   横轴Recall、纵轴PPV；AUPRC为曲线下面积，**正负样本不均衡时比AUC更可靠**，本项目核心指标。

## 指标联动规律
阈值下调（放宽阳性判定）：$TPR\uparrow、FPR\uparrow、PPV\downarrow$（漏诊减少、误诊增多）
阈值上调（收紧阳性判定）：$TPR\downarrow、FPR\downarrow、PPV\uparrow$（漏诊增多、误诊减少）

# 六、阈值调优方法（推理阶段后处理，仅用验证集）
所有方法目标：从模型输出概率中，选取最优分界阈值划分正负样本；**阈值不影响AUC/AUPRC**。

## 1. Youden 指数（ROC曲线）
$J=TPR-FPR$，取J最大值对应阈值；追求整体区分能力最优，通用场景。

## 2. F1-max（PR曲线）
最大化F1分数，均衡PPV与Recall，**样本不均衡任务首选**。

## 3. ROC/PR 曲线人工约束选点
根据业务要求限定 FPR/TPR/PPV，在约束范围内择优阈值。

## 4. PRC-TPR（PR曲线，医疗场景核心）
以召回率TPR为硬性标准，优先保证高检出率、**严控漏诊**，贴合风险筛查临床需求。

## 实验规范（5折交叉验证）
1. 仅用**每折验证集**调阈值，训练集、独立测试集不参与；
2. 5折分别计算阈值，同方法取均值作为全局阈值；
3. 最终效果只用独立测试集评估，杜绝数据泄露。

# 七、高频面试追问&标准应答（精简版）
1. **5折交叉验证每一折是否训练新模型？**
   是。每一轮用4折训练独立新模型，1折做测试，最终取5组指标均值。

2. **PR/ROC曲线上的点和线段代表什么？**
   每个点对应一个分类阈值下的一组指标；线段代表阈值连续变化时指标的动态变化。多条曲线对应不同折次，重合度越高模型越稳定。

3. **为什么医疗任务优先看AUPRC而非AUC？**
   医疗数据正负样本不均衡，ROC曲线易虚高；PR曲线对样本失衡更敏感，AUPRC能真实反映模型对高风险样本的识别能力。

4. **阈值调优会不会造成过拟合？**
   阈值调优属于推理后处理，不更新模型权重；只要不用测试集调优，就不会过拟合。

5. **Cox模型为什么能处理截尾数据？**
   建模时标记截尾状态，完整利用随访时长信息，不会直接剔除未发病/失访样本。

# 八、整体知识逻辑串讲（面试自述使用）
我的项目是口腔种植风险多模态预测，使用ResNet提取影像特征并去掉FC层做特征输出，结合临床变量做乘法融合。
实验上采用患者级5折交叉验证防数据泄露，通过标签洗牌、消融、可解释性实验验证模型未走捷径、确实学到病灶相关特征。
评价体系上，因样本不均衡，重点使用AUPRC、Recall、PPV、F1；结合Youden指数、F1-max、PRC-TPR多种方式调优阈值，兼顾指标与临床防漏诊的需求。
同时我也了解生存分析领域的Cox模型，可将影像特征与临床变量结合，拓展到长期随访风险预测场景。



追问 1：训练和测试阶段行为不一样，会不会有问题？
答：不会。训练时做输出缩放补偿，就是为了对齐训练与推理的数据分布，保证两阶段输出期望一致，因此不会影响最终预测效果。
追问 2：卷积层为什么不用普通 Dropout？
答：卷积层负责提取图像边缘、纹理等空间特征，随机单点丢弃神经元会破坏局部特征完整性，降低特征提取质量。如果需要对卷积层正则，业内一般使用 DropBlock 等改进方案。
追问 3：Dropout 设太大 / 太小会怎么样？
答：丢弃概率太小，防过拟合效果弱；概率太大，大量神经元被关闭，网络特征学习不充分，容易出现欠拟合。工程上 FC 层常用 0.3~0.5。
追问 4：Dropout 和 数据增强、早停（Early Stop）作用区别？
答：三者都是防过拟合手段：数据增强扩充样本、增加数据多样性；早停在验证集指标下降时终止训练；Dropout 从网络结构层面正则化，实际项目中经常搭配使用。
